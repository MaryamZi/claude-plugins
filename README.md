# Claude Plugins

A collection of Claude Code plugins.

## Installation

```
/plugin marketplace add MaryamZi/claude-plugins
```

## Available Plugins

### [insights-to-automations](plugins/insights-to-automations/) — Insights to Automations

Turn your `/insights` report into skills, agents, hooks, and CLAUDE.md rules based on actual usage patterns.

```
/plugin install insights-to-automations@MaryamZi-claude-plugins
```

```
/insights-to-automations:run
```

See the [plugin README](plugins/insights-to-automations/README.md) for full details.
